# Лабораторная работа №6-7. Взаимодействие с пользовательским интерфейсом и State в Jetpack Compose. Написание автоматических тестов
Приложение для автоматического расчёта суммы чаевых на основе введённой суммы счёта и процента.

# Цель работы
Познакомиться с понятием состояния (state) в Android-приложениях и научиться
использовать его в интерфейсе с помощью Jetpack Compose. В результате
лабораторной работы будет создано приложение калькулятор чаевых (Tip Time),
которое автоматически рассчитывает сумму чаевых. Изучить автоматизированные
тесты в Android.

# Функциональность
- Ввод суммы счёта
- Ввод процента чаевых
- Автоматический расчёт суммы чаевых в реальном времени
- Переключатель округления суммы чаевых вверх
- Поддержка альбомной ориентации с прокруткой
- Иконки в полях ввода
- Кнопки действий на клавиатуре (Next / Done)
- Локальный тест расчёта чаевых
- Инструментальный UI-тест через Jetpack Compose

# Технологии
- Kotlin
- Jetpack Compose-декларативный UI
- Material Design 3-компоненты интерфейса
- State/remember/mutableStateOf-управление состоянием
- State Hoisting-подъём состояния
- JUnit-локальные тесты
- Compose UI Test-инструментальные тесты
