# Лабораторная работа №5. Jetpack Compose: контейнеры, ARGB-модель и Material Design
# Цель работы:
Повторить основы разработки пользовательского интерфейса на
Jetpack Compose, работу с контейнерами Row и Column, познакомиться с ARGBмоделью цветов, компонентом Scaffold и принципами Material Design.
# Краткое описание приложения
Создано простое приложение на Jetpack Compose.
1. Реализован главный экран с использованием:
   * Column как основного контейнера
   * Row для горизонтального расположения элементов
   * Box для отображения цветных блоков
2. На экране присутствуют:
   * заголовок из двух строк текста
   * горизонтальный ряд из трёх цветных квадратов
   * один квадрат должен быть полупрозрачным
   * один контейнер с вложенными Row и Column
3. Цвета заданы в ARGB-формате.
4. Код разбит на отдельные @Composable функции.
